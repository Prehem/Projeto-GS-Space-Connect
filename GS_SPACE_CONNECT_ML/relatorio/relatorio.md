# Relatório Técnico

## Problema
Eventos climáticos extremos representam riscos ambientais e econômicos.

## Objetivo
Criar um modelo de Machine Learning capaz de prever eventos extremos utilizando dados climáticos.

## Metodologia
- Coleta de dados
- Limpeza
- Análise exploratória
- Treinamento do modelo
- Avaliação de métricas

## Tecnologias
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Resultados
O modelo apresentou boa taxa de acurácia para previsão de eventos extremos.

## Conclusão
Machine Learning pode auxiliar sistemas espaciais e ambientais na previsão e mitigação de riscos.
