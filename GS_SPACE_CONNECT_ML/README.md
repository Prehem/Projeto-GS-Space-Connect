# GS Space Connect - Machine Learning

## Objetivo
Desenvolver uma solução de Ciência de Dados e Machine Learning aplicada ao tema da Global Solution.

## Tema
Previsão de eventos climáticos extremos utilizando dados climáticos inspirados em aplicações espaciais.

## Tecnologias
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Como executar

### Instalar dependências
```bash
pip install -r requirements.txt
```

### Abrir o notebook
```bash
jupyter notebook
```

## Estrutura do Projeto

- data → base de dados
- notebook → notebook principal
- relatorio → relatório técnico
- images → imagens e gráficos

## Resultados
O modelo RandomForest foi utilizado para prever eventos extremos climáticos.
